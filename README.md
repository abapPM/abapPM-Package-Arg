# apm Package Args

Parse the things that can be arguments to `apm install`.

apm is a *package manager* 📦 for ABAP applications and modules, a *website* 🌐, and a *registry* 📑.

You can find the *website* at https://abappm.com and the *registry* at https://registry.abappm.com.

NO WARRANTIES, [MIT License](https://github.com/abapPM/abapPM-Package-Arg/blob/main/LICENSE)

## Usage

Not documented yet

## Prerequisites

SAP Basis 7.50 or higher

## Installation

Install `@apm/package-arg` as a global module in your system using [apm](https://abappm.com).

or

Specify the `@apm/package-arg` module as a dependency in your project and import it to your namespace using [apm](https://abappm.com).
